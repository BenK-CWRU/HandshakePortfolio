This repository contains some of the work I have done using simulation software and EDAs such as LTSpice and KiCad.

The FinalSchem_rev7 folder contains the KiCad project files for the amplitude shift keying and frequency shift keying demodulator circuit.

The ASKFSKDemodulatorFinalSchematic.asc file is the LTSpice file to simulate the amplitude shift keying and frequency shift keying demodulator.

DP2OpAmp.asc is an op amp design I made in LTSpice.

LDO is a low dropout linear regulator design I made in LTSpice.

OscillatorFunctionGen.asc is a function generator design I made in LTSpice.

PWMtoOptoDevice.asc is a PWM generator based on an external signal that is coupled to an optoelectronic device to output the same PWM signal. I designed this in LTSpice.

kHzClockGen.asc is an astable oscillator circuit that was made to generate a 1 kHz clock signal, which I designed in LTSpice.

I also have more experience with Autodesk Fusion, but most of the work is closely related to lab work that has yet to be patented.
Without disclosing the whole work, I have designed a circuit that contains a microcontroller which periodically polls an on-board pressure sensor.
It contained wireless power harvesting circuitry, so it could be powered with an inductive link to a source coil.
It was a two-layer board with tight trace routing. I researched the tolerances with our PCB vendor before layout to ensure I could design the smallest board possible.
