This repository contains some of the work I have done using simulation software and EDAs such as LTSpice and KiCad.

I also have more experience with Autodesk Fusion, but most of the work is closely related to work that has yet to be patented.
Without disclosing the whole work, I have designed a circuit that contains a microcontroller which periodically polls an on-board pressure sensor.
It contained wireless power harvesting circuitry, so it could be powered with an inductive link to a source coil.
It was a two-layer board with tight trace routing. I researched the tolerances with our PCB vendor before layout to ensure I could design the smallest board possible.
